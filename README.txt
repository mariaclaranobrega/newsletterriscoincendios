Newsletter - Risco de Incendios

1 - Extração de dados através de API (http://api.ipma.pt/)
2 - Exibição de dados em página web em formato Newsletter (framework: Flask)
3 - Periodizador de tarefas para que a API seja atualizada a cada Xseg, a cada refresh da página (biblioteca: threading)
4 - Envio de HTML por email (biblioteca: SMTP)
5 - Agendamento do envio do email (biblioteca: schedule)
